# verify-leyu-notes

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存放，不依赖后端服务，可直接通过浏览器打开查看。

## 项目简介

`verify-leyu-notes` 是一个静态 HTML 页面归档仓库。仓库中的每个页面均为独立文件，内容涵盖笔记、说明、工具界面等常见用途。本项目不针对任何特定域名或网站，仅作为通用归档空间，方便页面内容的保存与分发。

## 目录说明

```
.
├── README.md          # 本文件
└── pages/             # 存放所有 HTML 页面
    ├── example1.html
    ├── example2.html
    └── ...
```

- `pages/` 目录下为所有已归档的 HTML 文件，文件名反映页面内容或用途。
- 页面之间相互独立，无交叉依赖关系。

## 页面归档说明

- 每个 HTML 页面均为自包含文件，内部包含完整的样式与脚本。
- 页面可由用户直接下载或通过 GitHub Pages 等功能在线访问。
- 归档内容以笔记、工具、说明类页面为主，不包含交互式后台或动态数据请求。

## 维护说明

- 新增页面请放入 `pages/` 目录，并确保文件名简洁明了。
- 不建议在页面中嵌入外部资源（如图片、脚本），以保持归档的独立性和稳定性。
- 如需修改或删除页面，请直接操作对应文件。

## 使用方式

1. 克隆本仓库到本地。
2. 在浏览器中打开 `pages/` 目录下的任意 HTML 文件。
3. 也可通过 GitHub Pages 功能部署后在线访问。
